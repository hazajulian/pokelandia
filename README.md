# Preview

PokéLandia is a modern Pokédex focused on clean design, responsive layouts and reusable React components.

The application allows users to browse Pokémon, explore them by type, view detailed information, save favorites and switch between light and dark themes.

# Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/pokelandia.git
```

Go to the project

```bash
cd pokelandia
```

Install dependencies

```bash
npm install
```

Create the environment file

```env
VITE_CONTACT_EMAIL=your@email.com
```

Start the development server

```bash
npm run dev
```

---

# Build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

# Environment Variables

```env
VITE_CONTACT_EMAIL=your@email.com
```
